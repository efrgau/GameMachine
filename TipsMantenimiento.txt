Mantenimiento manual

1. Archivos temporales (%TEMP%)
	paso 1: abrir ejecutar (Windows+R)
	paso 2. debo escribir %TEMP% y darle enter
	paso 3. se abrio el explorador de carpetas en AppData/Local/Temp
	paso 4. selecciono todo con las teclas ctrl+E
	paso 5. presiona las teclas shift + suprimir para eliminar todos los archivos de forma permanente.

2. claves de registros
	paso 1. abrir ejecutar (Windows+R)
	paso 2. debo escribir prefetch
	PASO 3. se abrio el explorador en la carpeta C:/Windows/Prefetch
	paso 4. selecciono todo con las teclas ctrl+E
	paso 5. presiona las teclas shift + suprimir para eliminar todos los archivos de forma permanente.

MIRAR CLAVES DE REGISTROS
	paso 1. abrir ejecutar (Windows+R) 
	paso 2. escribir la palabra regedit
	paso 3. se abre el editor de registros
	paso 4. evidenciar las categorias de registros

3. Manejo de eventos (SO todo lo que hace necesita eventos)
	paso 1. abrir el panel de control
	paso 2. doble clic en herramientas administrativas
	paso 3. doble clic en el visor de eventos

4. desinstalación de programas
